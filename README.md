### how to use

- `> composer require`
- `> php spark db:create posmajoo`
- change emv.example to .env
- `> php spark migrate`
- `> php spark serve`
- admin @ /admin [admin:password]

### Techstack

- PHP 7.4 [CI 4]
- mysql 8
- CSS3 [BS-4]
- js [jquery 4.3.1]
